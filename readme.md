# Fibonacci sequence


### [npm link](https://www.npmjs.com/package/fibonacci_sequence).

#### How to install
###### With npm:
```javascript
npm install --save fibonacci_sequence
```

###### With yarn:
```javascript
yarn add fibonacci_sequence
```

#### How to use:
```javascript
import { fibonacciSequence } from 'fibonacci_sequence';
const fib = fibonacciSequence(10)
```
